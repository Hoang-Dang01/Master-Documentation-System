# MDS Solo View (Góc Nhìn Tinh Gọn Hàng Ngày)

> **MDS — Master Documentation System**
> *Developer-AI Collaboration Projections*

---

## VIEW A — Lọc Theo Vai Trò (By Role View)

Chọn vai trò bạn đang đóng để tập trung vào các thư mục liên quan:
*   [PM (Project Manager)](../roles/pm) -> Lộ trình, ngân sách, phân công.
*   [BA (Business Analyst)](../roles/ba) -> Business Rules, Use Cases.
*   [SA (System Analyst)](../roles/sa) -> SRS, Domain Model.
*   [ARCH (Architect)](../roles/arch) -> Quyết định ADR, bảo mật.
*   [BE (Backend Dev)](../roles/be) -> Database, API Contract, Service.
*   [FE (Frontend Dev)](../roles/fe) -> UI Specs, state flow.
*   [QA (Quality Assurance)](../roles/qa) -> Test Plan, Test Cases.
*   [DEVOPS (Platform Ops)](../roles/devops) -> Deployment, giám sát.

---

## VIEW B — Lọc Theo Giai Đoạn (By Lifecycle View)

Xem các tài liệu cần thiết dựa trên phase hiện tại của dự án:
*   [Phase 0: Intake](../lifecycle/00_intake) -> Yêu cầu ban đầu.
*   [Phase 1: Discovery](../lifecycle/01_discovery) -> BRIEF, CONSTRAINTS.
*   [Phase 2: Analysis](../lifecycle/02_analysis) -> Yêu cầu nghiệp vụ REQ.
*   [Phase 3: Design](../lifecycle/03_design) -> DB DDL, API Contract, ADR.
*   [Phase 4: Planning](../lifecycle/04_planning) -> Lộ trình & Phân rã công việc.
*   [Phase 5: Implementation](../lifecycle/05_implementation) -> Mã nguồn BE/FE.
*   [Phase 6: Testing](../lifecycle/06_testing) -> QA Test Cases, Bugs.
*   [Phase 7: Deployment](../lifecycle/07_deployment) -> IaC, Dashboards.
*   [Phase 8: Operations](../lifecycle/08_operations) -> Giám sát SRE & SLO.
*   [Phase 9: Evolution](../lifecycle/09_evolution) -> Bảo trì & Tối ưu hóa.
