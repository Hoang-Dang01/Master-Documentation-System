# MDS Solo View (Góc Nhìn Tinh Gọn Hàng Ngày)

> **MDS — Master Documentation System**
> *Developer-AI Collaboration Projections*

---

## VIEW A — Lọc Theo Vai Trò (By Role View)

Chọn vai trò bạn đang đóng để tập trung vào các thư mục liên quan:
*   [PM (Project Manager)](../10_ROLES/PM) -> Lộ trình, ngân sách, phân công.
*   [BA (Business Analyst)](../10_ROLES/BA) -> Business Rules, Use Cases.
*   [SA (System Analyst)](../10_ROLES/SA) -> SRS, Domain Model.
*   [ARCH (Architect)](../10_ROLES/ARCH) -> Quyết định ADR, bảo mật.
*   [BE (Backend Dev)](../10_ROLES/BE) -> Database, API Contract, Service.
*   [FE (Frontend Dev)](../10_ROLES/FE) -> UI Specs, state flow.
*   [QA (Quality Assurance)](../10_ROLES/QA) -> Test Plan, Test Cases.
*   [DEVOPS (Platform Ops)](../10_ROLES/DEVOPS) -> Deployment, giám sát.

## VIEW B — Lọc Theo Giai Đoạn (By Lifecycle View)

Xem các tài liệu cần thiết dựa trên phase hiện tại của dự án:
*   [Phase 1: Discovery](../20_LIFECYCLE/01_DISCOVERY) -> BRIEF, CONSTRAINTS.
*   [Phase 2: Analysis](../20_LIFECYCLE/02_ANALYSIS) -> Yêu cầu nghiệp vụ REQ.
*   [Phase 3: Design](../20_LIFECYCLE/03_DESIGN) -> DB DDL, API Contract, ADR.
*   [Phase 4: Implementation](../20_LIFECYCLE/04_IMPLEMENTATION) -> Mã nguồn BE/FE.
*   [Phase 5: Testing](../20_LIFECYCLE/05_TESTING) -> QA Test Cases, Bugs.
*   [Phase 6: Deployment](../20_LIFECYCLE/06_DEPLOYMENT) -> IaC, Dashboards.
