# Database Sharding Pattern

Mẫu phân mảnh cơ sở dữ liệu (Database Sharding) nhằm chia nhỏ dữ liệu từ một bảng lớn ra nhiều máy chủ database vật lý khác nhau.

## Các phương pháp Sharding
1.  **Horizontal Sharding (Range-based / Hash-based)**: Phân chia theo dòng dữ liệu.
2.  **Vertical Sharding**: Phân chia theo cột dữ liệu.
